# Remove Products From Cart — удаление товара из корзины

1. Открыть `/products`
2. Добавить товар в корзину
3. Нажать "View Cart"
4. **Ожидание:** `/view_cart`

[main.cartContent]
5. **Ожидание:** таблица с товарами видна
6. Нажать кнопку удаления (`X`) для товара
7. **Ожидание:** товар удалён из корзины
