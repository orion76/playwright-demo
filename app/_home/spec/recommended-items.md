# Recommended Items — добавление в корзину из рекомендованных товаров

1. Открыть `/`

[main.featuresItems]
2. Прокрутить вниз до блока "recommended items"
3. **Ожидание:** заголовок "recommended items" виден
4. Нажать "Add To Cart" на рекомендованном товаре
5. Нажать "View Cart"
6. **Ожидание:** URL `/view_cart`
