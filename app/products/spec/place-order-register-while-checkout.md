# Place Order: Register while Checkout — оформление заказа с регистрацией при checkout

1. Открыть `/products`

[main.productList]
2. **Ожидание:** заголовок "All Products" виден
3. Добавить товары в корзину
4. Нажать "Cart"
5. **Ожидание:** URL `/view_cart`
6. Нажать `checkoutBtn`
7. **Ожидание:** модальное окно с предложением зарегистрироваться
8. Нажать "Register / Login"
9. **Ожидание:** URL `/login`
