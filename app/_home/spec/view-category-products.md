# View Category Products — просмотр товаров по категориям

1. Открыть `/`

[sidebar.category]
2. **Ожидание:** заголовок "Category" виден
3. Нажать `women`
4. Нажать на подкатегорию "Dress" в разделе Women
5. **Ожидание:** URL `/category_products/*`
6. **Ожидание:** заголовок "WOMEN - TOPS PRODUCTS" виден
7. Нажать `men`
8. Нажать на подкатегорию в разделе Men
9. **Ожидание:** URL `/category_products/*`
